# 🚔 MONITORING-S-CURITAIRE-GRAND-LOM- - Analyse des Interventions de Police

Projet d’analyse de données sur la **sécurité publique** dans le Grand Lomé, axé sur l’optimisation de la réactivité des forces de l’ordre.

## 🎯 Objectifs du Projet
- Analyser la répartition spatio-temporelle des incidents
- Mesurer les temps d’intervention
- Identifier les zones à haut risque
- Proposer des recommandations pour améliorer la sécurité

## 🛠️ Solutions Mises en Œuvre
- Nettoyage des données d’incidents
- Création d’un **dashboard Power BI** interactif
- Analyses spatiales et temporelles
- Rédaction d’un rapport complet avec dictionnaire de données

## 📁 Fichiers du Projet

| Fichier                                      | Description |
|---------------------------------------------|-----------|
| `incidents_grand_lome.csv`                  | Données des incidents |

| `Contexte.docx`                             | Document de contexte |

## 🛠️ Technologies
- **Power BI** (Dashboard & Visualisations)
- **Power Query** (Nettoyage)
- Analyse spatio-temporelle
